# Blinkit Sales Analysis Dashboard with Power BI

## Introduction

The Blinkit Sales & Customer Insights Dashboard is designed to help retailers and business decision-makers analyze **sales performance**, **product categories**, and **customer satisfaction** using a structured set of visual insights.

Using a real world retail dataset, this dashboard highlights how factors such as **fat content, product type, outlet scale, and location** influence total sales and customer perception.

Whether you're analyzing business health or exploring retail optimization strategies, this dashboard simplifies the process by providing a **single, interactive reporting experience**.
